디버깅을 하려면 가장 먼저 디버깅 하려는 위치에 중단점을 설정해야한다.
프로그램이 실행되는 도중 멈추고자 하는 코드 부분에 F9클릭 or 마우스 클릭

원하는 위치에 디버깅이 끝나면 설정된 위치에서 사용되는 변수값 확인 가능
프로그램이 어떻게 동작하고 있는지 파악 가능
버그가 발생한 위치와 문제점을 쉽게 파악 가능

중단점 이후 디버깅 F11(한 단계씩 코드 실행) or F10(프로시저 단위) 
함수 다음 코드 실행하고싶으면 Shift + F11을 누른다
즉 눈에 보이는 코드를 한 줄씩 실행하는게 아니라 내부적으로 실행하는 모든코드를
실행하는 것

커서까지 실행 Ctrl + F10을 누른다

실행될 문 변경 드래그를 하면된다. 단, 실행된 부분의 결과는 사라지지 않고 그대로 있음

포인터 변수 값 확인 조사식 선택 '변수 + 위치, 보고싶은 행 개수'

중단점 조건 / 참인경우 변경된 경우 적중횟수 맞춰서 설정하면 된다.

중단점 비활성화 Ctrl + F9를 누른다

디버깅 중 직접 코딩하지 않고 어떤 결과 나오는지 알고 싶을 땐 직접실행을 하면 된다.
직접 실행 창에서 변수값을 변경하면 바로 적용이 된다. 원래 변수값은 그대로이다.
